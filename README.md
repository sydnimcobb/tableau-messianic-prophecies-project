# tableau-messianic-prophecies-project
# About This Project
Tableau dashboard visualizing Messianic prophecies in the biblical canon, connecting Old Testament prophecies to New Testament fulfillments. This was a beginner project (first-time tableau user) created from a self-made dataset. The narrative and presentation give greater explanation on challenges in both the process and execution.

- 'Final Project DATA 605-1.twb' - Tableau dashboard
- 'Final Project DATA 605-1.doc' - Narrative and methodology
- 'Final Project Data 605.ppt' - PowerPoint presentation
