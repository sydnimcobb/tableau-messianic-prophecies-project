# tableau-messianic-prophecies-project
# Overview
Tableau dashboard visualizing Messianic prophecies in the biblical canon, connecting Old Testament prophecies to New Testament fulfillments. Includes dashboard, PowerPoint presentation, and narrative of the project and processes.

# About This Project
This was a beginner project (first-time tableau user) created from a self-made dataset. The narrative and presentation give greater explanation on challenges in both the process and execution.
